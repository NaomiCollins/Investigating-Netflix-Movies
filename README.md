🎬 Investigating Netflix Movies 

📌 Project Overview

Netflix! What started in 1997 as a DVD rental service has since exploded into one of the largest entertainment and media companies in the world.
With a vast collection of movies and series available on the platform, this project presented a perfect opportunity to apply exploratory data analysis (EDA) techniques within the entertainment industry.
Using the dataset netflix_data.csv, I explored movie trends from the 1990s, focusing on nostalgic styles and uncovering insights about this iconic movie decade.

🎯 Project Objectives

Analyze Netflix movie data specifically from the 1990s decade (starting from 1990).
Identify the most frequent movie duration during this period.
Define and analyze short action movies (movies under 90 minutes).
Count the number of short action movies released in the 1990s.

🔎 Approach

Filtered the dataset to include only movies released between 1990–1999.
Performed exploratory data analysis to determine the most common movie duration (saved as duration).
Defined short movies as films under 90 minutes.
Counted the number of short action movies in the 1990s (saved as short_movie_count).
